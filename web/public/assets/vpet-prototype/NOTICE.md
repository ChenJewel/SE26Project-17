# VPet Temporary Prototype Assets

This folder contains a limited subset of the built-in VPet animation assets for a non-commercial school-course prototype.

Source: https://github.com/LorisYounger/VPet

The VPet project code is licensed under Apache-2.0. The built-in animation/image assets have separate attribution requirements in the VPet README. This prototype documents the source in `web/docs/11-pet-companion.md` and must not sell or redistribute these animation files as a paid asset package.

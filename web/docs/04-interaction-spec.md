# 04 交互规格

## 注册与登录

- 未登录时只显示注册/登录页，不显示底部导航和主应用浮层。
- 登录和注册支持切换。
- 邮箱输入后显示校园认证提示。
- 密码支持显示/隐藏。
- 登录或注册成功后进入首页。
- 我的页展示当前登录邮箱、认证状态，并提供退出登录。
- 设置页退出登录操作会回到注册/登录页。

## 全局桌宠

- 登录并完成资料后，桌宠作为全局悬浮层出现在主应用页面。
- 桌宠默认使用最小尺寸 `sm`，可在状态面板中切换 `SM / MD / LG`。
- 轻触桌宠触发摸头动作和随机台词；拖动桌宠移动位置；长按或拖动时显示捏/拎起动作。
- 拖动松手时，只有手指/指针基本触碰到屏幕左右边缘，才触发收边探头。
- 收边探头状态只保留较明显的小人露出面积；“点我 / 拖我”提示只显示几秒，后续只保留探头。
- 从探头状态触碰或拖动桌宠时，会优先恢复到桌面内，避免再次误收边。
- 点击 `Lv / 心情` 状态条打开桌宠面板；面板可拖动，且桌宠本体层级高于面板。
- 侧边快捷按钮包含投喂、衣柜预留、收起和显隐开关；隐藏后只保留小眼睛按钮在桌宠正下方。
- 面板支持喝水、思考、睡觉、爬墙、停止爬墙、说话表情、移动预览和回到桌面。
- 如果用户约 5 分钟没有与桌宠互动，且桌宠空闲、未贴边、未爬墙、未睡觉，桌宠会低概率自动播放几秒移动小动作并冒泡提示。

## 首页

- 标签筛选支持多选，不限制数量。
- 点击 `全部` 清空已选标签。
- 折叠态标签只展示一行，保证卡片是首页视觉主体。
- 左滑上一张，右滑下一张。
- 滑动时下层卡片浮现，释放后目标卡从下层浮起覆盖当前卡。
- 点击“想一起吃”：
  - 自动生成交换约饭卡请求。
  - 自动进入对应用户聊天详情。
  - 聊天详情中展示系统交换卡片。

## 发卡片

- 可以填写昵称、文案、日期、时间、地点、人数和标签。
- 标签可以从已有标签中选择，也可以创建新标签。
- 卡片预览头像可以选择。
- 点击发布后：
  - 新卡片写入当前用户 `userId`、昵称、头像和认证状态。
  - 新卡片插入首页卡片流，所有用户可见。
  - 新标签写入全局标签池。
  - 新卡片同步出现在“我的”页“我发布的约饭卡”。
  - 原型使用浏览器 `localStorage` 保留发布结果，刷新页面后仍可查看。
  - 返回首页。
- 支持选择可见范围：同校可见、关注可见、仅匹配推荐。
- 支持保存草稿的原型反馈，正式版应进入草稿箱接口。

## 全局搜索

- 可搜索用户、约饭卡片和帖子。
- 点击用户打开用户主页详情。
- 用户主页中，非本人用户显示“关注”按钮。
- 用户主页展示关注数、粉丝数、匹配度、共同偏好，以及私信/发起约饭/屏蔽/举报等原型入口。
- 关注用户后，同步到“我的”页关注用户列表，并同步到消息页“新增关注”入口。
- 用户主页里的发布帖子和约饭卡片可以继续点开具体详情。
- 点击约饭卡打开约饭卡详情。
- 点击帖子打开帖子详情。
- 从搜索结果进入详情后，关闭详情返回搜索浮层，而不是直接回社区或首页。

## 社区

- 可浏览推荐/关注/附近/餐厅/生活/经验频道。
- 可发布文字、照片、视频类型帖子。
- 发帖编辑器支持公开/仅关注/仅同校可见范围。
- 发帖编辑器支持保存草稿的原型反馈。
- 发布帖子后立即打开新帖子详情。
- 点帖子详情里的作者头像可打开作者主页。
- 图文帖子如果是照片，可点开查看大图。
- 视频帖子打开沉浸式视频详情，不显示示例平台标签。
- 评论 sheet 支持查看、发布、喜欢、收藏、举报原型操作。

## 消息

- 点击底部导航“消息”永远进入消息列表。
- “赞和收藏”入口显示数量，点开可查看谁赞/收藏了哪个帖子，并可进入帖子详情。
- “新增关注”入口显示新增关注数量，点开可查看关注用户列表，并可进入用户主页。
- “评论和@”入口显示数量，点开可查看回复/@列表，并可进入相关帖子评论区。
- 点击消息列表中的会话进入聊天详情。
- 聊天详情顶部头像可进入对方用户主页。
- 从“想一起吃”进入消息时，只自动打开对应聊天详情一次。
- 聊天详情左上角返回回到消息列表。
- 系统交换卡片消息包含：
  - 用户自己的约饭卡。
  - 点击卡片可查看约饭卡详情。
  - `拒绝（不好意思下次哦）`
  - `聊聊看`
- 点击交换按钮后，显示双方可见的状态结果。

## 我的

- 头像可点开查看，并在原型中切换头像字符。
- 我的偏好是常用标签，点击编辑可选择/创建标签。
- 我发布的帖子可点开详情。
- 最近创作的划卡可点开约饭卡详情。
- 我发布的评论会跳转到相关帖子的评论区。
- 喜欢/收藏的帖子可点开详情。
- 关注用户可点开其他用户主页。
- 搜索或其他页面关注的新用户会同步出现在关注用户列表。

## 设置

- 设置入口位于“我的”页面右上角。
- 设置页采用浅色风格，与原项目 UI 保持一致。
- 设置列表项可进入二级详情。
- 设置二级详情中的每个条目可打开操作面板，展示当前状态、说明和继续配置入口。
- 注销账号属于危险操作，必须从设置列表进入确认面板二次确认；确认后调用后端删除云端账号数据并自动退出登录。
- 当前移除了不必要项，例如多语言翻译、收货地址、未成年人模式、新功能体验、存储空间等。

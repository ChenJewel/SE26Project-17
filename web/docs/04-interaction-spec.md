# 04 交互规格

## 首页

- 标签筛选支持多选，不限制数量。
- 点击 `全部` 清空已选标签。
- 折叠态标签只展示一行，保证卡片是首页视觉主体。
- 左滑上一张，右滑下一张。
- 滑动时下层卡片浮现，释放后目标卡从下层浮起覆盖当前卡。
- 点击“想一起吃”：
  - 自动生成交换约饭卡请求。
  - 自动进入对应用户聊天详情。
  - 聊天详情中展示系统交换卡片。

## 发卡片

- 可以填写昵称、文案、日期、时间、地点、人数和标签。
- 标签可以从已有标签中选择，也可以创建新标签。
- 卡片预览头像可以选择。
- 点击发布后：
  - 新卡片插入首页卡片流。
  - 新标签写入全局标签池。
  - 返回首页。

## 全局搜索

- 可搜索用户、约饭卡片和帖子。
- 点击用户打开用户主页详情。
- 点击约饭卡打开约饭卡详情。
- 点击帖子打开帖子详情。
- 从搜索结果进入详情后，关闭详情返回搜索浮层，而不是直接回社区或首页。

## 社区

- 可浏览推荐/关注/附近/餐厅/生活/经验频道。
- 可发布文字、照片、视频类型帖子。
- 发布帖子后立即打开新帖子详情。
- 图文帖子如果是照片，可点开查看大图。
- 视频帖子打开沉浸式视频详情，不显示示例平台标签。
- 评论 sheet 支持查看、发布、喜欢、收藏、举报原型操作。

## 消息

- 点击底部导航“消息”永远进入消息列表。
- 点击消息列表中的会话进入聊天详情。
- 从“想一起吃”进入消息时，只自动打开对应聊天详情一次。
- 聊天详情左上角返回回到消息列表。
- 系统交换卡片消息包含：
  - 用户自己的约饭卡。
  - `拒绝（不好意思下次哦）`
  - `聊聊看`
- 点击交换按钮后，显示双方可见的状态结果。

## 我的

- 头像可点开查看，并在原型中切换头像字符。
- 我的偏好是常用标签，点击编辑可选择/创建标签。
- 我发布的帖子可点开详情。
- 最近创作的划卡可点开约饭卡详情。
- 我发布的评论会跳转到相关帖子的评论区。
- 喜欢/收藏的帖子可点开详情。
- 关注用户可点开其他用户主页。

## 设置

- 设置入口位于“我的”页面右上角。
- 设置页采用浅色风格，与原项目 UI 保持一致。
- 设置列表项可进入二级详情。
- 当前移除了不必要项，例如多语言翻译、收货地址、未成年人模式、新功能体验、存储空间等。

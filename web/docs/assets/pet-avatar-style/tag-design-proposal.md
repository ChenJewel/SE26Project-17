# 约饭App桌宠兴趣标签设计方案

为了让桌宠能够直观地展示用户的兴趣爱好，我们设计了一套**“兴趣贴纸/气泡”**体系。这些元素将以Q版、简约的风格呈现，装饰在桌宠的脸部、头发或周围空间，既美观又能快速识别用户标签。

## 1. 核心标签设计示例

| 兴趣分类 | 贴纸/元素设计 | 视觉呈现建议 |
| :--- | :--- | :--- |
| **摄影** | 复古小相机 | 放在脸颊一侧，像个小纹身 |
| **骑行** | 简约自行车或头盔 | 作为发饰（发卡）固定在头发上 |
| **美食家** | 冒热气的饭团、拉面或奶茶 | 悬浮在嘴角附近的小气泡里 |
| **音乐** | 灵动的音符或头戴式耳机 | 耳机可以直接“戴”在桌宠头上 |
| **游戏** | 像素风手柄或小红心 | 贴在额头一角或作为瞳孔的高光变形 |
| **电影** | 电影胶片或爆米花桶 | 环绕在头部的半透明装饰带 |
| **旅行** | 纸飞机或小行李箱 | 像云朵一样漂浮在头顶上方 |
| **运动** | 燃烧的小火焰或网球/足球 | 放在脸颊的腮红位置，增加动感 |

## 2. 视觉布局策略

*   **脸颊装饰（Cheek Decor）：** 适合小型、扁平化的图标，如相机、红心、星星。
*   **发饰替换（Hair Accessory）：** 适合具有结构感的物体，如自行车、乐器、水果。
*   **悬浮气泡（Floating Bubbles）：** 适合展示“当前想吃”或“动态爱好”，如饭团、火锅。
*   **腮红叠加（Blush Overlay）：** 将图标半透明处理，融入角色的腮红中，风格更统一。
